#  Book Recommender System

A Machine Learning based Book Recommendation System built using Python and Flask.

##  Features

- Recommend books based on user selection
- Popular books homepage
- Search by book title
- Similar book recommendations
- Simple and responsive web interface

##  Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS
- Bootstrap

##  Project Structure

```
Book-Recommender-System/
│
├── app.py
├── requirements.txt
├── books.pkl
├── popular.pkl
├── pt.pkl
├── similarity_scores.pkl
├── templates/
│   ├── index.html
│   └── recommend.html
└── README.md
```

##  How to Run

1. Clone the repository

```
git clone https://github.com/mana157/Book-Recommender-System.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```

4. Open your browser

```
http://127.0.0.1:5000
```
